📊 Excel Import Data (Data Structure)

(TH) การเตรียมไฟล์ Excel: จัดเรียงข้อมูล 3 คอลัมน์หลัก คือ หมวดหมู่, ชื่อรายการ และราคา เพื่อนำเข้าข้อมูลปริมาณมากในครั้งเดียว

(EN) Excel Preparation: Organize data into 3 columns: Category, Item Name, and Price for bulk importing.

Category (หมวดหมู่) Item Name (ชื่ออาหาร) Price (ราคา)
จานเดียวข้าวกะเพราหมูสับ60
ของว่างปอเปี๊ยะทอด50
ยำยำวุ้นเส้น70
เครื่องดื่มชาเย็น40

* รองรับไฟล์นามสกุล .csv หรือ .xlsx ตามโครงสร้างด้านบน

📌 About App

A button-based sales/expense recorder. Ideal for businesses with fixed menus. Features automatic total calculation and supports up to 10 concurrent queues/tables.

🎮 How to Use

Memory Management

  • Switch between 10 slots for different queues.
  • "Summary" button to view totals and print.

Operation Modes

Add Mode (+) Tap to add items.
Sub Mode (−) Tap to reduce or add discounts.
